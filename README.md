# Descrición

Aplicación que permite gestionar productos y registrar ventas en una tienda de abarrotes.


## Tecnologías utilizadas

- Java 17
- Spring Boot 3
- Spring Data JPA
- H2 Database (en memoria)
- Maven
- Lombok
- JUnit 5 + Mockito (para pruebas unitarias)
